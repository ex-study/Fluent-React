# 전문가를 위한 리액트
## 스터디 규칙
### 폴더 & 파일 구조
- **폴더명**: 해당 장의 숫자만 표기
  - 예시: `01장 - 입문자를 위한 리액트` → 폴더명은 `01`
- **파일명**: 본인 이름으로 `.md` 파일 작성
  - 예시: `이효린.md`, `변수미.md`

### 제출 규칙
- **마감 시간**: 스터디 당일 24:00
- **주의**: 준비를 했더라도 마감 시간을 넘기면 미제출로 간주
- **페널티**: 미제출 시 스터디원들에게 커피 사주기 ☕
- 
### Git 규칙
- **브랜치명**: `{장 번호}/{본인 이름}`
  - 예시: `01/hyorin`, `02/sumi`
  - Base 브랜치: `main`

- **PR 제목**: `[{장 번호}장] {챕터명}`
  - 예시: `[01장] 입문자를 위한 지식`
- **머지 방식**: Squash merge

### 최종 디렉토리 구조
```
.
├── 01/
│   ├── 이효린.md
│   ├── 변수미.md
│   └── 심여은.md
├── 02/
│   ├── 이효린.md
│   └── ...
└── README.md
```
